# 👑 Drag Race Pokédex

A **Pokédex-style explorer for RuPaul’s Drag Race contestants**.

This project aims to provide a **simple interface to browse, sort, and filter queens** from across the Drag Race universe — a bit like a Pokédex, but for drag legends.

Because let’s be honest:  
sometimes you _know_ the queen… but you can’t remember **which season she was on**.

---

## ✨ Features

Current and planned features include:

- 📋 **Large interactive table** of Drag Race contestants
- 🔎 **Search** queens by name
- 🔽 **Sort** by different attributes (season, franchise, etc.)
- 🎛️ **Filter** contestants by season or franchise
- 📱 Simple and lightweight UI focused on browsing the dataset

Future ideas:

- 🧠 **Drag Race quizzes**
- 🎲 Random queen generator
- 📊 Stats and fun trivia
- 👑 "Who's that queen?" game mode
